Glossary

Attribute
Attributes provide additional information about HTML elements.
  • class
    The class attribute specifies one or more class names for an HTML element. The class name can be used by CSS and JavaScript to perform certain tasks for elements with the specified class name.
  • href
    Links tell the browser where to go using an href attribute, which stores a URL.
  • id
    An HTML element can have an id attribute to identify it. Id elements should always be unique to that single element, and each element should never have more than one id.
  • src
    Specifies the location (URL) of the external resource.
Body
The body is the container for all of a page's content. Comes after the tag, within the overall tag.
Button
This tag defines a clickable button. Inside a button element you can put content, like text or images. This is the difference between this element and buttons created with the input element.
Children
An element that is an immediate descendent of another element or nested within another element is called a child.
CSS
Cascading Style Sheets is a declarative language that controls how webpages look in the browser.
CSS Cheatsheet
A complete listing pf all the properties, selectors types and allowed values.
CSS Layout
  • Box Model
    Is the specification that defines how a box and its attributes relate to each other. It consists of: margins, borders, padding, and the actual content.
  • Flexbox
    Method of positioning elements in horizontal or vertical stacks.
  • Float
    The float property is used for positioning and layout on web pages. It can only have 4 values: left, right, none or inherit.
  • Grid
    Method of using a grid concept to lay out content, providing a mechanism for authors to divide available space for layout into columns and rows using a set of predicatable sizing behaviors.
CSS3 Transitions
CSS Transitions allow property changes in CSS values to occur smoothly over a specified duration.
Debugging
Is the process of finding and fixing software coding errors.
Div
A block level container (or 'division' of the web page) for content with no semantic meaning.
Favicons
Also known as a shortcut icon or URL icon, is a file containing one or more small icons, associated with a particular website or web page.
Form
An HTML form contains form elements. Form elements are different types of input elements, like text fields, checkboxes, radio buttons, submit buttons, and more.
Head
Tag that surrounds important content that is invisible to the user, but is important to the browser. Elements within this tag contain metadata about the page and links to stylesheets, scripts, etc.
Headings
Heading elements like h1, h2, h3, ... allow you to use six levels of document headings, ranging from largest to smallest.
HTML
HyperText Markup Language is the most basic building block of the Web. It describes and defines the content of a webpage.
HTML Layout Elements
  • aside
    Defines content aside from the content (like a sidebar)
  • footer
    Defines a footer for a document or a section
  • header
    Defines a header for a document or a section.
  • nav
    Defines a container for navigation links.

HTML tag
All HTML files live within an over-arching html tag. This is the basic tag that defines an html document.
Hypertext
Is text that contains links to other texts.
Input
This tag specifies an input field where the user can enter data. Input elements are used within a form element
JS
JavaScript is a high-level, dynamic, weakly typed, prototype-based, multi-paradigm, and interpreted programming language. It's used to program the behavior of web pages.
Link
Links are found in nearly all web pages. Links allow users to click their way from page to page.
Lists
HTML supports two kinds of lists: ordered (denoted by the ol tag) and unordered (denoted by the ul tag).
Media Queries
Are useful when you want to apply CSS styles depending on a device's general type, specific characteristics, or environment.
Microinteractions
Are the little details that separate something you tolerate from something you love.
Properties
Ways in which you can style a given HTML element.
Pseudo Class
A pseudo-class is used to define a special state of an element.
Responsive Web Design (RWD)
Is about using HTML and CSS to resize, hide, shrink, enlarge, or move the content to make it look good on any device(desktops, tablets, and phones).
Rules for CSS
An at-rule is a CSS statement beginning with an at sign, '@', followed by an identifier and includes everything up to the next semi-colon, ';', or the next CSS block, whichever comes first.
Sandbox
A testing environment that isolates untested code changes and outright experimentation from the production environment or repository.
Script
This tag is used to define a client-side script (JavaScript). The script element either contains scripting statements, or it points to an external script file through the src attribute.
Selector
Is the HTML element name at the start of the rule set. It selects the element(s) to be styled.
Semantic
In programing, Semantics refers to the meaning of a piece of code.
Syntax
In linguistics, it's the set of rules, principles, and processes that govern the structure of sentences in a given language, specifically word order and punctuation.
Table
Is an element for displaying information in rows and columns. It divides information into rows (denoted by the tr tag) which contain cells (denoted by the td tag).
Tag/Element
Tags are basic labels that define and separate parts of your markup into elements. They are comprised of a keyword surrounded by angle brackets <>.
Text Editors
A type of program used for editing plain text files.
Title
This tag tells the browser what to display as the page title at the top and tells search engines what the title of your site is. It goes inside tags.
Validator
The Validation Service was created by the W3C to help Web developers validate CSS.
WHATWG
The Web Hypertext Application Technology Working Group is a community of people interested in evolving HTML and related technologies.
W3C
The World Wide Web Consortium is the main international standards organization working in the development of standards for the World Wide Web.
