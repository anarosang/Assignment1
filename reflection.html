Reflection

During this first term of learning web development, I was anxious to have already learned a bit and did not like it but as I was reviewing some topics and learning some new ones I started getting more interested in this assignment. I think web development is simple if we start with the more basic things instead of starting to write code without a concrete idea.

Through the lessons, I realized that one of my big problems was not making a sketch of what I wanted and, so I wasted more time. After realizing this the first thing I did for this assignment was some wireframe sketches. Although the final design of the site was as I wanted, it was a bit difficult to make it equally appealing on all the devices, another thing that I began to appreciate after the web development classes.
